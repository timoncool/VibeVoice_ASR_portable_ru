# VibeVoice ASR Portable - Русскоязычная версия

Портативная версия VibeVoice ASR для Windows 11 с полной русификацией интерфейса.

VibeVoice ASR - это современная модель распознавания речи от Microsoft, которая преобразует аудио в текст с высокой точностью и поддержкой диаризации спикеров.

## Возможности

- Распознавание речи в текст с высокой точностью
- Поддержка диаризации спикеров (определение кто говорит)
- Временные метки для каждого сегмента речи
- Поддержка 4-bit квантизации для экономии видеопамяти
- Полностью русскоязычный интерфейс
- Темная тема интерфейса
- Автоматическое открытие в браузере
- Портативность - все файлы в одной папке
- Поддержка множества аудио и видео форматов через FFmpeg

## Системные требования

- Windows 10/11 (64-bit)
- NVIDIA GPU с поддержкой CUDA (рекомендуется 8GB+ VRAM)
- Или CPU (медленнее, но работает)
- 16GB+ RAM
- 10GB свободного места на диске

### Рекомендуемые видеокарты

| Серия | CUDA версия | Flash Attention 2 |
|-------|-------------|-------------------|
| GTX 10xx (Pascal) | CUDA 11.8 | Нет |
| RTX 20xx (Turing) | CUDA 11.8 | Нет |
| RTX 30xx (Ampere) | CUDA 12.6 | Да |
| RTX 40xx (Ada Lovelace) | CUDA 12.8 | Да |
| RTX 50xx (Blackwell) | CUDA 12.8 | Да |

## Установка

1. Скачайте или клонируйте репозиторий
2. Запустите `install.bat`
3. Выберите вашу видеокарту из списка
4. Дождитесь завершения установки

Установщик автоматически скачает и настроит:
- Python 3.12.8 (портативный)
- PyTorch с поддержкой CUDA
- FFmpeg (портативный)
- Все необходимые зависимости
- Flash Attention 2 (для RTX 30xx/40xx/50xx)

## Запуск

1. Запустите `run.bat`
2. Приложение автоматически откроется в браузере
3. Выберите модель и нажмите "Загрузить модель"
4. Загрузите аудио файл или запишите с микрофона
5. Нажмите "Распознать речь"

## Модели

### Полная модель (microsoft/VibeVoice-ASR)
- Лучшее качество распознавания
- Требует ~8GB VRAM
- Рекомендуется для RTX 30xx и выше

### 4-bit модель (scerz/VibeVoice-ASR-4bit)
- Экономит видеопамять (~4GB VRAM)
- Немного медленнее
- Подходит для видеокарт с меньшим объемом памяти

## Поддерживаемые форматы

Благодаря встроенному FFmpeg поддерживаются все популярные форматы:

**Аудио:** MP3, WAV, FLAC, OGG, M4A, AAC, WMA, AIFF, и другие

**Видео:** MP4, MKV, AVI, MOV, WebM, и другие (извлекается аудио дорожка)

```
VibeVoice_ASR_portable_ru/
├── app.py              # Основное приложение
```

## Изоляция

Приложение полностью изолировано и не создает файлы за пределами своей папки:
- Все временные файлы в `temp/`
- Все модели в `models/`
- Все кэши в `cache/`
- Системные переменные окружения не изменяются

## Решение проблем

### Ошибка "CUDA out of memory"
- Используйте 4-bit модель
- Закройте другие приложения, использующие GPU
- Уменьшите параметр "Максимум токенов"

### Модель загружается медленно
- При первом запуске модель скачивается (~5GB)
- Последующие запуски будут быстрее (модель кэшируется)

### FFmpeg не найден
- Запустите `install.bat` повторно
- Или скачайте FFmpeg вручную с https://ffmpeg.org и распакуйте в папку `ffmpeg/`

### Ошибка при распознавании
- Проверьте формат аудио файла
- Попробуйте конвертировать в WAV
- Убедитесь, что аудио содержит речь
